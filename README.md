### 🧠 About Me

I'm a **Senior Software Engineer, Solution Architect, and Machine Learning Engineer** with over **9 years of experience** designing and deploying distributed systems and scalable AI pipelines.  
<br>
Currently at **[Baubap](https://www.baubap.com)** (YC S21), I lead MLOps and infrastructure initiatives — bridging the gap between **machine learning**, **software engineering**, and **cloud architecture** to build robust, production-grade AI systems.

#### 🔹 What I Do
- ⚙️ **MLOps Architect** — CI/CD for ML with **MLflow**, **Airflow**, and **AWS**
- ☁️ **Cloud Infrastructure** — Serverless, event-driven systems & data replication
- 🧩 **Distributed Computing** — Scalable analytics with **Dask**, **Lambda**, and **S3**
- 🧠 **AI Systems Design** — From experimentation to real-time deployment
- 🧱 **Architecture** — End-to-end ownership: data, backend, and automation

#### 🏆 Key Achievements
- ⚡ Built a real-time **data replication system** syncing 600+ tables and 30TB+ data with <30 min latency, zero downtime  
- 🚀 Architected a **CI/CD pipeline for ML models** using MLflow, reducing deployment times from **days to minutes**  
- 🧮 Scaled distributed processing with **Dask + AWS**, cutting job times from **days to hours**  
- 🔐 Delivered secure, fault-tolerant systems using **event-driven and serverless** architectures
